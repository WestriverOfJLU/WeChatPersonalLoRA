# Personal Facts Example

Copy this file to `data/personal_facts.md` and edit it with facts that are safe and accurate for your own assistant.

```text
# Personal facts

- Name:
- Education:
- Major / field:
- Current role:
- Known constraints:

# Notes

Only write facts you are comfortable using locally. Do not put private chat logs, contact names, phone numbers, addresses, tokens, or credentials here.
```
